## 9. REVIEW NOTES (4000 characters max)

Hello App Review Team,

Gourdo is a grocery management app designed to help users reduce food waste and save money through intelligent expiration tracking, AI-powered recipe suggestions, and expert produce guidance.

**Key Features:**
1. Barcode scanning (via device camera and Open Food Facts API)
2. OCR-based date extraction (using on-device processing)
3. Produce freshness database with 38 common items
4. AI recipe generation (Claude API calls via Supabase Edge Functions)
5. Push notifications (with user opt-in via notification preferences)
6. Cross-device sync (cloud storage via Supabase)

**Data Privacy:**
- User pantry data is encrypted and stored in Supabase
- API key for Claude never reaches the client device (server-side only)
- All user data is deletable via account deletion
- Subscription data managed securely via RevenueCat

**Third-Party Services:**
- Supabase (Database, Auth, Edge Functions)
- Open Food Facts API (Barcode product lookup)
- Anthropic Claude API (Recipe generation)
- RevenueCat (Subscription management)

**Compliance:**
- Full GDPR and CCPA compliance
- Privacy policy available in-app and at gourdo.app/privacy
- Terms of Service at gourdo.app/terms
- No user data sold to third parties

**Testing Credentials:**
- Sandbox App Store Connect account setup and tested
- Free tier and Pro trial modes fully functional
- No external payment processing within this submission

**Notes for Reviewers:**
- App requires iOS 14.0 or later
- Barcode scanning requires camera permission (user-prompted)
- Push notifications require explicit user opt-in
- Some features (OCR, AI recipes) require an internet connection

If you have any questions or require additional information, please reach out to support@gourdo.app.

Thank you for reviewing Gourdo!
